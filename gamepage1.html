神魂系统

神魂获得方式：
新手上线默认赠送1点神魂点 // 每天势力战奖励50点神魂点 // 使用游戏内签到系统每次签到奖励5点神魂点
每晚22:00 - 23:00 全图开启神魂争夺战 - 成功击杀拥有神魂点超过50点玩家1次 获得10点神魂点/被击杀损失10点神魂点

神魂查看方式：
拥有神魂BUFF后上线会有BUFF提醒,以及属性加成
游戏中打命令 !信息 可查看当前神魂点数

神魂BUFF/属性加成
神魂 普通BUFF/属性加成 需要神魂点：5 0 点 属性增幅： 攻击力 增加 50 攻击/ 防御力 增加 50 攻击/ 生命力 增加 100 / 内功力 增加 100
神魂 高级BUFF/属性加成 需要神魂点：200点 属性增幅： 攻击力 增加 100 攻击/ 防御力 增加 100 攻击/ 生命力 增加 200 / 内功力 增加 200
神魂 史诗BUFF/属性加成 需要神魂点：400点 属性增幅： 攻击力 增加 300 攻击/ 防御力 增加 300 攻击/ 生命力 增加 500 / 内功力 增加 500
神魂 传说BUFF/属性加成 需要神魂点：800点 属性增幅： 攻击力 增加 500 攻击/ 防御力 增加 500 攻击/ 生命力 增加 1000 / 内功力 增加 1000
神魂兑换元宝

神魂兑换元宝方式：
游戏中打命令进行兑换，例如游戏中打命令： !神魂10兑换元宝 系统奖励：1000元宝

神魂兑换元宝命令及兑换比例：
!神魂10兑换元宝 = 1000元宝
!神魂50兑换元宝 = 5000元宝
!神魂100兑换元宝 = 10000元宝
!神魂300兑换元宝 = 30000元宝
!神魂500兑换元宝 = 50000元宝
!神魂1000兑换元宝 = 100000元宝

幸运女神 - (每晚20:00/21:00/22:00开启)

幸运女神简介说明：
每晚20:00/21:00/22:00开启 系统自动抽取当前在线玩家一名，幸运女神降临，发放幸运奖励,请多留意游戏内系统公告
每天三个时段发放奖励，此功能-只针对综合大区进行开放

幸运女神奖励如下：
元宝：500 / 武勋：1000 / 积分：500 / 神魂点：5
宝箱：【上古黑盒】

新门派战

参加方式：
每晚21:00开启门派战
门派帮主需要召集所有参加 门派玩家 通过韦大宝 移动至【银币市场】输入命令 !申请门战
在银币市场等待 门战开始自动传送至门派战地图
注意：离开 银币市场 系统默认视为不参加门战。
注意：银币市场 内本帮派人员不得低于 8 人否则无法参加门战。
注意：门派击杀人数要超过50,不然不会获得胜利门派失败门派奖励。

门派战奖励：
胜利门派全员奖励：1000积分
胜利门派全员奖励：【上古黑盒】
胜利门派门派武勋奖励：1点

失败门派全员奖励：500积分
失败门派随机奖励：斗神炫彩丹BUFF/5小时/加成
失败门派门派武勋减少：1点

门派BUFF/属性加成
门派1级BUFF/属性加成 需要门派武勋：5 0 点 属性增幅： 攻击力 增加 50 攻击/ 防御力 增加 50 攻击/ 生命力 增加 50 / 内功力 增加 50
门派2级BUFF/属性加成 需要门派武勋：100点 属性增幅： 攻击力 增加 100 攻击/ 防御力 增加 100 攻击/ 生命力 增加 100 / 内功力 增加 100
门派3级BUFF/属性加成 需要门派武勋：200点 属性增幅： 攻击力 增加 200 攻击/ 防御力 增加 200 攻击/ 生命力 增加 200 / 内功力 增加 200
门派4级BUFF/属性加成 需要门派武勋：400点 属性增幅： 攻击力 增加 400 攻击/ 防御力 增加 400 攻击/ 生命力 增加 400 / 内功力 增加 400
门派5级BUFF/属性加成 需要门派武勋：600点 属性增幅： 攻击力 增加 500 攻击/ 防御力 增加 500 攻击/ 生命力 增加 500 / 内功力 增加 500

门派BUFF捐献：
门派内所有成员可自由捐献门派武勋

命令：!门派捐献5000积分 = 门派武勋增加5点
命令：!门派捐献10000积分 = 门派武勋增加10点
命令：!门派捐献50000积分 = 门派武勋增加50点
命令：!门派捐献100000积分 = 门派武勋增加100点

注意：捐献人物如果退出门派/解散门派/踢出门派/捐献积分不予返还
